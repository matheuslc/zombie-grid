Zombie Grid

A Sass SIMPLE GRID SYSTEM

  • How much columns you like
  • Fluid, but if you like fixed
  • Really simple
  • Code reuse with @extend

    Getting Started

  • @import grid to your style file.
  • @include zombie-grid mixin with properties.
  • @extend %zombie- columns properties.
